# rustcommon-logger

The `rustcommon-logger` crate provides a basic logger implementation

## Overview

This crate provides a basic logger implementation which logs to stdout. It also
provides a `fatal!` macro for logging fatal errors and terminating execution.

## Getting Started

### Building

rustcommon is built with the standard Rust toolchain which can be installed and
managed via [rustup](https://rustup.rs) or by following the directions on the
Rust [website](https://www.rust-lang.org/).

#### View library documentation
